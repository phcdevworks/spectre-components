# Codex Workspace Files

This directory contains OpenAI Codex support files for
`@phcdevworks/spectre-components`.

Claude Code remains the lead developer and `CLAUDE.md` remains the
authoritative implementation guide. Codex uses these files for documentation,
release readiness, production stabilization, repo hygiene, changelog/release
note support, and config standardization.

## Files

- `change-watch.md` - lightweight branch or release-candidate review log.
- `release-readiness.md` - Codex release and production handoff checklist.

## Related Root Files

- `CODEX.md` - Codex operating guide.
- `CODEX_REVIEW_CHECKLIST.md` - component change review checklist.
- `CODEX_RELEASE_CHECKLIST.md` - release candidate checklist.
